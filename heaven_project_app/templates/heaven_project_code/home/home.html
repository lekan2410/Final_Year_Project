{% blocktrans trimmed asvar intro_text_main %} Welcome to the Haven Website! This platform is designed to help users find nearby services, or search for specific services they need. It is primarily aimed at individuals experiencing homelessness and other vulnerable users who may be looking for support. {% endblocktrans %} {{ intro_text_main }}

{% trans "Image By" %} Paulo Silva (Unsplash)

{% trans "Why this was created?" as header_1_text %} {{ header_1_text }}

{% blocktrans trimmed asvar intro_text_secondary %} Homelessness remains a global issue, and its rising rates are concerning for those affected who may not know where to find support. This platform was created to address this need by providing spaces for people experiencing homelessness or vulnerability to access support. It’s designed to alleviate some of the hardship and distress they face. {% endblocktrans %} {{ intro_text_secondary }}

{% trans "Image By" %} Youssef Naddam (Unsplash)

{% trans "What the platform offers?" as header_2_text %} {{ header_2_text }}

{% blocktrans trimmed asvar intro_text_services %} Users can find nearby services such as food banks, shelters, and other support options using filters on the services page. By clicking on a service card, users can view more details, including its location and an integrated map. Once signed in, users can also leave reviews for each service. A chatbot is available to answer questions about the platform or help users find the nearest services. {% endblocktrans %} {{ intro_text_services }}
